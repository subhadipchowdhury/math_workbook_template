\section{Basic notation}
The class provides the common number systems
\[
\N\subseteq \Z\subseteq \Q\subseteq \R\subseteq \C,
\qquad
\text{and a generic field } \F.
\]

It also includes several special alphabets and notational helpers:
\[
\U,
\qquad
\P(\N),
\qquad
\O(n^2).
\]

The Lie-theoretic and affine-space symbols can be used separately as follows:
\[
\u,
\qquad
\A^2.

\section{Delimiters, arrows, and elementary symbols}
\[
\la u,v \ra = u_1v_1+u_2v_2,
\qquad
\ceil*{\vf{7}} = 1,
\qquad
\floor*{\vf{7}} = 0.
\]

\[
x_n \rh x,
\qquad
\mu_n \xrh \mu.
\]

\[
A = \{x\in\R \suchthat x^2\le 1\}.
\]

\[
\phi(x)=x^2+\epsilon,
\qquad
\cis \theta = \cos\theta + i\sin\theta.
\]

\[
\grad f = \left(\frac{\d f}{\d x},\frac{\d f}{\d y}\right),
\qquad
\bar{A}=\left[-1,1\right].
\]

\section{More vector notation}
Let $\vec{v}=\langle 1,2\rangle$ and $\uvec{e}_1=\langle 1,0\rangle$.

The reciprocal notation can also be combined with vector expressions:
\[
\vf{2}\vec{v}.
\]

For emphasis, one can also write the bold-affine symbol separately as
\[
\A,
\]
without mixing it with several other math alphabets in the same formula.
